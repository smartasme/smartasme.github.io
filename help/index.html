Q&A and tips

How to set voice (TTS) engine:

In Settings, select one of the TTS engines. If only default Pico TTS available, we recommend to install Google Text to Speech engine or other similar products for better voice quality.

Long hold the TTS engine, to configure the actual TTS engine

Each category's title is clickable to collapse or uncollapse the corresponding settings

Click "Alarms auto close in" expand the settings, input the minutes for auto closure for all alarms.

Hold "+" (quick one-off alarm)

Tips for Alarm Detail screen

Click the back arrow button on the top to cancel the changes

Drag to change the sequence

Click to turn the TTS volume off

Click to turn the sound off

Why no sound:

Sound is muted in global settings

Voice (TTS) is turned off in global Settings

Voice (TTS) is turned off individually

Enable link clicks & search a keyword

Click "Enable links" to make the links clickable on the screen

Click magnifier icon to search a keyword, and press it again to search next occurance.

Fail to load actions

Some action requires network: like weather, transport and news

Check if network is available, for custom news link, it needs to be RSS2.0 format.
